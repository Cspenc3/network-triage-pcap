# OSI vs TCP/IP Model

## OSI Model (7 Layers)
1. Application
2. Presentation
3. Session
4. Transport
5. Network
6. Data Link
7. Physical

## TCP/IP Model (4 Layers)
1. Application
2. Transport
3. Internet
4. Network Access

## Key Notes
- OSI is theoretical; TCP/IP is practical (real-world).
- Example mapping: HTTP (Application), TCP (Transport), IP (Internet), Ethernet (Link/Physical).
